# -WXWork-Robot
本程序是用Qt框架开发，用于对企业微信中的群聊机器人进行控制

目前第一版本仅支持纯文本格式和markdown格式的通知发送，以及@指定人 或者@所有人

程序开发环境是Qt 5.12.5 msvc2017_64bit
